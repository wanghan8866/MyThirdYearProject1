## Requirements
### Installation
1. To install all required libraries
```pip install -r requirements.txt```
2. To get the code 
```git clone https://github.com/wanghan8866/MyThirdYearProject1.git```
3. To run the GUI
   1. ```cd <PATH TO MyThirdYearProject1>```
   2. ```python main.py ```
